# Portfolio (React + Vite)

Recreation of the provided portfolio site using React, Vite, TypeScript, and Tailwind CSS.

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`

## Notes

- Tailwind config: `tailwind.config.js`
- Global styles: `src/index.css`
- Main layout: `src/App.tsx`
